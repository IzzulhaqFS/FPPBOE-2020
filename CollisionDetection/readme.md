---------- COLLISON DETECTION ----------

Deskripsi

Collision Detection adalah cara untuk mengetahui apakah dua objek dalam layar telah bertabrakan. Biasanya digunakan dalam banyak game arcade. Aplikasi ini akan mengimplementasikan Collision Detection pada game Alien Shooter.

Cara Kerja

Setiap objek akan dibugkus dalam persegi panjang. Ketika dua persegi panjang berbenturan maka program akan mendeteksi bahwa telah terjadi collision dan memicu event yang bersangutan, seperti game over atau berhasil menembak sebuah alien.
